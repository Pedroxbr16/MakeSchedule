# Monta Escala

Aplicacao web para montar escala com preview em tempo real.

## Funcionalidades

- Campo geral de `titulo`
- Campo geral de `data`
- Campo geral de `observacoes`
- Tabela de escala no formato `pessoa | funcao`
- Adicionar e remover linhas dinamicamente
- Preview instantaneo da escala
- Exportacao do preview em `PNG`

## Rodando localmente

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`.
